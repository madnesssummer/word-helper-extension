单词助手插件测试页面

测试说明

这个页面用于测试单词助手插件的功能。请按照以下步骤进行测试：

  1. 在浏览器中加载插件
  2. 选中下面的英文单词进行划词测试
  3. 查看是否显示查询次数
  4. 点击收藏按钮测试收藏功能

测试单词

请尝试选中以下单词来测试插件功能：

  • hello - 一个简单的问候语
  • world - 世界
  • computer - 计算机
  • programming - 编程
  • javascript - JavaScript编程语言
  • extension - 扩展程序
  • translation - 翻译
  • dictionary - 字典

测试文本段落

The computer is a powerful tool for programming and development. Many developers use javascript to create interactive web applications. A browser extension can provide additional functionality like translation services or act as a digital dictionary. When you select a word, the plugin should show the query count and allow you to save it to your vocabulary book.
